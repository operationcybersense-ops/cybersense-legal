# cybersense-legal
1. Introduction This privacy policy governs the use of the Cybby and CyberQuest browser extensions created by Operation Cybersense.

2. Data Collection & Cloud Storage To provide synchronization and features across devices, these extensions utilize Supabase (a backend-as-a-service provider).

Authentication: If you sign in, we store your email address to manage your account.

Functional Data: We store [e.g., user preferences, quest progress, or safety settings] within our Supabase database. This data is used solely for the functionality of the extension.

No Tracking: We do not collect browsing history, keystrokes, or any data outside the scope of the extension's specific features.

3. Data Security Your data is protected using Supabase’s security infrastructure, including Row Level Security (RLS) to ensure that only you can access your own data.

4. Third-Party Services We use Supabase for data storage. You can view their privacy commitment at Supabase Privacy Policy. We do not sell or share your data with any other third parties.

5. User Rights You may request the deletion of your account and associated data at any time by contacting us through the Operation Cybersense GitHub organization.
